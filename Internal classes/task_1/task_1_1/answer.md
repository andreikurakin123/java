1. PublicInnerClass:
Этот класс доступен из любого места, когда у вас есть экземпляр внешнего класса Main. Мы создаем экземпляр публичного внутреннего класса и вызываем метод show().

2. PrivateInnerClass:
Этот класс доступен только внутри внешнего класса Main. Мы создаем метод createPrivateInner(), который создает экземпляр PrivateInnerClass и вызывает его метод show(). Этот метод можно вызывать из main.
